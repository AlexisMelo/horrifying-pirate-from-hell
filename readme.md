##### POUR LANCER LE JEU #####

Lancer le programme " Menu principal "
On peut accéder aux autres programmes depuis celui ci

Le programme "JEU1" est commenté partiellement
les programmes JEU2,JEU3,JEU4 sont seulement des
dérivés du programme JEU1 avec des copié/collé pour 
ajouter des joueurs

Le programme "config2" est la page de configuration entre
le menu et le plateau
